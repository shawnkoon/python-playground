Python playground applications.

## How to run

1. Install Python 3.

2. Run either the following.

```bash
位 python3 -m <folder_name> (e.g python3 -m hello_world)

位 python3 -m <folder_name>.<file_name> (e.g python3 -m hello_world.app)
```

3. Run unit tests to see the test coverage

```bash
位 python3 -m unittest discover

or

位 make test
```
